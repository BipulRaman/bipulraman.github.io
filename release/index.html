Release Notes:

[ Saturday - April 10, 2021 ]

Today I am pleased to announce the general availability of the refreshed version of my website www.bipul.in. I have passionately designed and coded it by myself. This website will be the node for all of my digital presence. The previous version of the website was a blog site, which has been moved to blogs.bipul.in. The refreshed website will have pages with various content which includes my work, experiences, micro-blogs, link to my blog site, contact, appointment scheduling etc.

Technical Design:

The website has been developed on MicroServices Architecture. At the frontend - ReactJs, GatsBy, Material-UI has been used for rapid and robust UI-UX development. At the backend - Azure Function has been used for creating APIs on .Net Core in C#. Data are stored in Azure CosmosDB and Azure Storage. Frontend of the website has been deployed to GitHub Pages. Continuous Integration and Continuous Deployment (CI-CD) has been implemented with GitHub Actions. The Azure Active Directory has been used to manage the user authentication on this website. Cloudflare has been used to manage the firewall to prevent attacks on the website.

If you come across any issues, please report it on GitHub Repository.
